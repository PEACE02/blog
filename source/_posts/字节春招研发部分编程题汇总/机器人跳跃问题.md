## 公式推导

```C++
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int n, h;
    // long long 只能过 3 个样例，float 能过 7 个，double 全过
    double E = 0.0;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> h;
        E += h * pow(2, n - i);
    }
    cout << ceil(E / pow(2, n)) << endl;
    return 0;
}
```
pow(x, y) 的时间复杂度为 log(y)
